# Blinkit Marketing Performance Dashboard

This repository contains a complete Excel-based *Marketing Performance Dashboard* created to analyze Blinkit's digital marketing campaigns.  
The dashboard provides interactive insights using PivotTables, PivotCharts, KPI cards, and slicers for dynamic filtering.
